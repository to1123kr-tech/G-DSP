G-DSP 개발기록

DAY 1 · 2026.03.18

프로젝트 시작

G-DSP는 토목·측량·인허가 업무를 하나의 웹 플랫폼에서 처리하기 위해 시작된 프로젝트이다. 여러 사이트를 이동해야 하는 기존 업무 방식을 개선하고, 지도 조회 · 토지정보 조회 · 도면 수량 산출 · 민원 관리 등을 하나의 시스템에서 처리하는 것을 목표로 개발을 시작하였다.

오늘 구현된 기능

  • 로그인 화면 (현재는 빈칸으로 바로 입장)
  • 지번 입력 → 용도지역 / 지목 / 건폐율 자동 표시
  • 검색 시 지도 위치 자동 이동
  • 지도 서비스 탭 전환
  • 외부 행정 사이트 바로가기
  • DXF 도면 수량 자동 산출
  • 민원 접수번호 관리
  • 담당자 연락처 검색
  • 작업 이력 자동 저장
  • 문의 글 등록 기능

지도 시스템

  • VWorld 일반지도 / 위성지도
  • 카카오 일반지도 / 위성지도 / 로드뷰
  • 네이버 지도
  • 국토정보 플랫폼

지도 전환을 한 화면에서 바로 사용할 수 있도록 구성하였다.

DXF 도면 기능

  • .dxf 파일 업로드
  • 레이어별 수량 계산
  • 색상 분류
  • mm / m 단위 전환
  • CSV 저장
  • PDF 인쇄

현재 DXF 기능은 정상 작동 상태이다.

업무 관리 기능

담당자 연락처

  • 총 138명 등록
  • 검색 기능
  • 전화번호 클릭 통화

민원 접수 관리

  • 접수번호 등록
  • 접수중 / 보완중 / 완료 상태 관리
  • 10개 지역 관리

향후 개발 계획

  • 회원가입 시스템
  • 사용자별 데이터 저장
  • DB 서버 구축
  • 수치지형도 자동 다운로드
  • DWG 도면 지원
