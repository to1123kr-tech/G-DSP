⚠ 분할 모드 — 법적 고지 및 이용 동의
본 시스템의 면적 계산 결과는 공식 측량 성과가 아닙니다.
인허가 신청·등기·계약·공문서 등에 직접 사용을 금지합니다.
정확한 면적은 반드시 공인 측량사를 통해 확인하시기 바랍니다.
결과값 오류로 인한 모든 책임은 사용자 본인에게 있습니다.
측량·수로조사 및 지적에 관한 법률 제2조·제39조
⚙️ 자동 갱신 설정 방법
화성시 민원 사이트는 CORS 보안으로 브라우저에서 직접 조회할 수 없습니다.
아래 로컬 서버를 실행하면 자동 갱신이 활성화됩니다.
📋 설치 (최초 1회)
pip install flask requests beautifulsoup4
▶️ 실행 (매번)
python server.py
# 그 후 브라우저에서 http://localhost:5050 접속
💡 server.py 와 hwaseong_crawl.py 를 index.html 과 같은 폴더에 넣어주세요.
