G-DSP 크롬 확장 프로그램 (2종 공통)

요약 — 아래 확장 프로그램은 사용자의 개인정보를 수집하거나 외부로 전송하지 않습니다. 로그인 정보(아이디·비밀번호)를 저장하지 않으며, 모든 동작은 사용자의 브라우저와 사용자 본인 계정 안에서 이루어집니다.

대상 확장 프로그램

  • 국토정보플랫폼 수치지형도 다운로드 — 수치지형도(DXF) 검색·신청·다운로드 자동화
  • 산림공간정보 다운로드 — 임상도·산림입지토양도·산사태위험지도 신청·다운로드 자동화

두 확장은 서로 독립적으로 동작하며, 데이터를 공유하지 않습니다. 확장별로 다른 부분은 아래에 나누어 적었습니다.

1. 수집하는 개인정보 (공통)

두 확장 모두 개발자 또는 제3자에게 전송되는 개인정보를 수집하지 않습니다. 사용자를 식별할 수 있는 정보를 외부 서버로 보내지 않습니다.

2. 확장별 상세

국토정보플랫폼 수치지형도 다운로드 NGII

기기에 로컬로 저장되는 정보
사용 편의를 위해 다음 정보를 사용자 기기의 브라우저 저장소(chrome.storage.local)에 저장합니다.

  • 마지막으로 검색한 지번(주소)
  • 선택한 축척(1:5,000 / 1:1,000)
  • 선택한 사용 용도
  • 최근 실행 상태 및 시각

이 정보는 사용자 기기에만 저장되며 외부로 전송되지 않고, 확장을 삭제하면 함께 삭제됩니다.

로그인 정보
이 확장은 국토정보플랫폼 계정의 아이디·비밀번호를 저장하지 않습니다. 로그인은 사용자가 정부 통합로그인 화면에서 직접 수행하며, 확장은 사용자의 기존 로그인 세션을 이용해 절차를 이어갈 뿐입니다.

접근 권한
이 확장은 아래 사이트에서만 동작하며, 이는 수치지형도의 검색·신청·다운로드 절차를 자동화하기 위한 것입니다.

  • map.ngii.go.kr, nlippd.ngii.go.kr — 국토정보플랫폼(검색·신청·파일 전송)
  • gdsp.kr — 사용자의 다운로드 요청을 확장에 전달

산림공간정보 다운로드 FOREST

로그인 정보
산림공간정보서비스의 자료 신청은 별도 인증 절차가 없는 «일반인 신청»을 이용하므로, 이 확장은 아이디·비밀번호를 요구하지도, 저장하지도 않습니다.

사용자가 직접 하는 입력
신청서의 «신청목적»과 «활용용도 및 신청사유»는 확장이 임의로 입력하지 않습니다. 예시 버튼을 제공할 뿐이며, 사용자가 직접 확인하고 선택·작성해야 다음 단계로 진행됩니다.

접근 권한
이 확장은 아래 사이트에서만 동작하며, 이는 산림공간자료의 신청·다운로드 절차를 자동화하기 위한 것입니다.

  • map.forest.go.kr — 산림공간정보서비스(신청 양식 자동 입력·자료 다운로드)
  • gdsp.kr — 사용자의 «자동으로 받기» 요청(대상 주소)을 확장에 전달

3. 원격 코드 (공통)

두 확장 모두 원격 코드를 사용하지 않습니다. 모든 코드는 확장 패키지에 포함되어 있으며, 외부에서 코드를 내려받아 실행하지 않습니다.

4. 제3자 제공 및 판매 (공통)

두 확장 모두 사용자 데이터를 제3자에게 제공, 양도, 판매하지 않습니다.
